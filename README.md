# 价格单元

价格单元，英文：Price Unit，缩写：PU，主要用于 **CA3W 认证机构管理系统** 中符文的价格计算。

价格单元的定义和取值：[pu.md](pu.md)

---

## 2023年 1 PU = 34.75 元

```text
2018 271.03
2019 312.61
2020 387.53
2021 374.36
2022 392.12

(271.03 + 312.61 + 387.53 + 374.36 + 392.12) / 5 / 10 = 34.753
```

## 2022年 1 PU = 32.42 元

```text
2017 275.54
2018 271.03
2019 312.61
2020 387.53
2021 374.36

(275.54 + 271.03 + 312.61 + 387.53 + 374.36) / 5 / 10 = 32.4214
```
